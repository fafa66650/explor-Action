# Explor’Action multi-villages

Dépose directement le contenu de ce dossier dans un dépôt GitHub Pages.

## Contenu
- Argelès-sur-Mer
- Saint-André
- Laroque-des-Albères
- Villelongue-dels-Monts
- Collioure
- Port-Vendres
- Banyuls-sur-Mer

## Fonctionnalités
- mode solo / équipe
- chrono + bonus temps
- score par réponse, GPS, défi, photo et équipe
- validation GPS ou validation d'équipe
- photo-preuve sur certaines étapes
- stockage local automatique

## Important
La géolocalisation dépend du navigateur et de l'autorisation utilisateur.
En cas de souci terrain, utilisez le bouton "Valider avec l’équipe".
